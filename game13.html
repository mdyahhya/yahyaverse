What is Python?

Python is a powerful yet beginner-friendly programming language created by Guido van Rossum in 1991 [web:1]. Think of it as a way to talk to computers using words that are almost like English!

Why Learn Python?

Python is one of the most popular languages in 2025 because it's easy to read and write [web:9]. Unlike other languages that look like alien code, Python reads like a recipe or instructions!

Real-World Uses:

  • ü§ñ Creating AI and machine learning
  • üìä Analyzing data and making graphs
  • üéÆ Making games and animations

Your First Python Program üöÄ

The Classic "Hello World"

Every programmer's first program! Let's write it in Python:

print("Hello, World!")

That's it! Just one line and you're a programmer. Compare this to other languages that need 5-10 lines for the same thing [web:1].

How It Works:

  • print() - tells Python to show something
  • "Hello, World!" - the text to display
  • Parentheses () hold what you want to print
  • Quotes "" mark it as text (called a "string")

Try These:

print("Welcome to Python!") print("My name is Alex") print(2025)

Notice how you can print text OR numbers!

Variables: Storing Information üì¶

What are Variables?

Variables are like labeled boxes that store information [web:7]. You give them a name and put data inside!

name = "Sarah" age = 15 height = 5.6 is_student = True

Understanding Variables:

  • name - stores text (string)
  • age - stores whole numbers (integer)
  • height - stores decimals (float)
  • is_student - stores True/False (boolean)

The Magic Part:

In Python, you don't tell the computer "this is text" or "this is a number". Python figures it out automatically [web:6]! This is called dynamic typing.

x = 10 # x is a number x = "Hi" # now x is text!

Try changing variable values - Python adapts!

Data Types Explained üé®

Types of Data in Python:

1. Numbers (Numeric)

score = 95 # Integer price = 19.99 # Float complex_num = 2+3j # Complex

2. Text (String)

message = "Hello Python!" quote = 'Learning is fun'

3. True/False (Boolean)

is_raining = False has_license = True

4. Lists (Collections)

fruits = ["apple", "banana", "orange"] numbers = [1, 2, 3, 4, 5]

Lists store multiple items in order [web:7]. Use square brackets [].

Check Types:

print(type(95)) # int print(type(19.99)) # float

Python as a Calculator:

# Addition print(5 + 3) # Output: 8 # Subtraction print(10 - 4) # Output: 6 # Multiplication print(6 * 7) # Output: 42 # Division print(20 / 4) # Output: 5.0 # Integer Division print(17 // 5) # Output: 3 # Remainder (Modulo) print(17 % 5) # Output: 2 # Power print(2 ** 3) # Output: 8

Making Decisions with If ü§î

Conditional Statements:

Programs need to make decisions. Python uses "if" statements to check conditions [web:10].

age = 16 if age >= 18: print("You can vote!") else: print("Too young to vote")

Important! Notice the indentation (spaces before print). Python uses indentation instead of curly braces {} [web:1].

Multiple Conditions:

score = 85 if score >= 90: print("Grade: A") elif score >= 80: print("Grade: B") elif score >= 70: print("Grade: C") else: print("Grade: F")

Use elif (else if) for multiple checks [web:3].

Comparison Operators:

  • == (equal to)
  • != (not equal)
  • > (greater than)
  • < (less than)

Loops: Repeating Actions üîÑ

For Loops:

Need to repeat something? Use loops [web:3]!

# Print numbers 1 to 5 for i in range(1, 6): print(i) # Loop through a list fruits = ["apple", "banana", "cherry"] for fruit in fruits: print(fruit)

The range(1, 6) creates numbers from 1 to 5 (stops before 6).

While Loops:

count = 0 while count < 5: print(count) count = count + 1

While loops continue as long as the condition is True [web:3].

Pro Tip: Be careful with while loops! Make sure they eventually stop, or they'll run forever.

Breaking Out:

for i in range(10): if i == 5: break print(i)

Functions: Reusable Code üéØ

What are Functions?

Functions are reusable blocks of code with a name. Write once, use many times [web:3]!

def greet(name): print("Hello, " + name + "!") # Call the function greet("Alice") # Hello, Alice! greet("Bob") # Hello, Bob!

Breaking It Down:

  • def - defines a new function
  • greet - the function name
  • (name) - parameter (input)
  • Indented code runs when called

Functions with Return:

def add(a, b): return a + b result = add(5, 3) print(result) # Output: 8

The return statement sends a value back [web:3].

Functions make code organized and easier to fix!
