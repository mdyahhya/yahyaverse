Welcome to Python!

What is Python?

Python is a powerful, high-level programming language created by Guido van Rossum and first released in 1991 [web:1]. It has become one of the most popular programming languages in the world, loved by beginners and professionals alike.

Why Learn Python?

Python is designed to be easy to read and write. Unlike many programming languages that use complex syntax with lots of brackets and semicolons, Python uses simple, English-like statements that make sense even to non-programmers [web:1].

What Can You Do With Python?

  • Build websites and web applications
  • Create artificial intelligence and machine learning models
  • Analyze data and create visualizations
  • Automate repetitive tasks
  • Develop games and desktop applications
  • Write scripts for system administration

Python works on all major platforms including Windows, Mac, Linux, and even mobile devices [web:1]. This cross-platform compatibility makes it an excellent choice for any project.

Your First Python Program

The Classic "Hello, World!"

Every programmer's journey begins with a simple program that displays text on the screen. In Python, this is incredibly straightforward [web:1]:

print("Hello, World!")

That's it! Just one line of code. The print() function tells Python to display whatever is inside the parentheses on the screen.

Understanding the Syntax

  • print - A built-in function that outputs text
  • () - Parentheses contain the argument (input) for the function
  • "Hello, World!" - A string (text) enclosed in quotes

Try These Examples:

print("Welcome to Python!") print("My name is Sarah") print(2025) print(3.14)

Notice how you can print both text (strings) and numbers. Python handles different data types seamlessly [web:16].

Comments in Python

Use the # symbol to add comments - notes that Python ignores [web:3]:

# This is a comment print("This runs") # Comment after code

Variables: Storing Data

What Are Variables?

Variables are named containers that store data in your computer's memory [web:13]. Think of them as labeled boxes where you can put information and retrieve it later.

Creating Variables

In Python, creating a variable is simple - just use the assignment operator (=) [web:13]:

name = "Alice" age = 25 height = 5.6 is_student = True

The variable name goes on the left, the equals sign in the middle, and the value on the right.

Variable Naming Rules

  • Must start with a letter or underscore (_)
  • Can contain letters, numbers, and underscores
  • Cannot use Python keywords (like print, if, for)
  • Case-sensitive: "Name" and "name" are different

Using Variables

greeting = "Hello" name = "Bob" message = greeting + " " + name print(message) # Output: Hello Bob

Variables make your code flexible and reusable [web:13]. You can change a variable's value and all code using it automatically updates!

Data Types in Python

Understanding Data Types

Python has several built-in data types to represent different kinds of information [web:16]. Let's explore the most important ones.

1. Numeric Types

integer_num = 42 # int float_num = 3.14 # float complex_num = 2 + 3j # complex
  • int - Whole numbers (positive or negative)
  • float - Decimal numbers
  • complex - Complex numbers with real and imaginary parts

2. String Type

text = "Hello Python" quote = 'Single quotes work too' multiline = """This spans multiple lines"""

Strings are sequences of characters enclosed in quotes [web:16]. You can use single, double, or triple quotes.

3. Boolean Type

is_valid = True is_empty = False

Booleans represent truth values - either True or False [web:16]. They're essential for decision-making in programs.

Conditional Statements

Making Decisions in Code

Programs need to make decisions based on conditions. Python uses if, elif, and else statements [web:12].

Basic If Statement

age = 18 if age >= 18: print("You are an adult") else: print("You are a minor")

Important: Notice the indentation (spaces before print). Python uses indentation to group code blocks [web:1].

Multiple Conditions

score = 85 if score >= 90: print("Grade: A") elif score >= 80: print("Grade: B") elif score >= 70: print("Grade: C") elif score >= 60: print("Grade: D") else: print("Grade: F")

The program checks each condition in order and executes the first one that's True [web:3].

Logical Operators

age = 25 has_license = True if age >= 18 and has_license: print("Can drive")

Loops in Python

Repeating Actions

Loops allow you to execute code multiple times without writing it repeatedly [web:12].

For Loops

Use for loops to iterate over sequences [web:13]:

# Loop through numbers for i in range(5): print(i) # Prints 0, 1, 2, 3, 4 # Loop through a list fruits = ["apple", "banana", "cherry"] for fruit in fruits: print(fruit)

The range() function generates a sequence of numbers [web:15]. range(5) creates numbers from 0 to 4.

While Loops

While loops continue as long as a condition is True [web:12]:

count = 0 while count < 5: print(count) count = count + 1 # Increment counter

Warning: Make sure while loops eventually become False, or they'll run forever!

Break and Continue

for i in range(10): if i == 5: break # Exit loop print(i)

Functions: Reusable Code

Creating Functions

Functions are reusable blocks of code that perform specific tasks [web:3]. They help organize your code and avoid repetition.

Defining Functions

def greet(name): print("Hello, " + name + "!") # Call the function greet("Alice") # Output: Hello, Alice! greet("Bob") # Output: Hello, Bob!

Function Components:

  • def - keyword to define a function
  • greet - function name
  • (name) - parameter (input)
  • Indented code is the function body

Returning Values

def add(a, b): return a + b result = add(5, 3) print(result) # Output: 8

The return statement sends a value back to the caller [web:3].

Multiple Parameters

def calculate(x, y, operation): if operation == "add": return x + y elif operation == "multiply": return x * y
